# 深空画布 · 项目记忆

## 项目是什么

AI 科普短视频自动生产线。输入 AI 图片 + JSON 脚本 → 输出小红书竖屏 MP4（1080×1920）。

## 当前状态

- **版本**: v3（FFmpeg zoompan 运镜，无文字叠加）
- **已完成选题**: 创生之柱（天文）、巨齿鲨（史前巨兽）、山海经异兽（神话）
- **管线可运行**: ✅ pipeline/v3 路径已去硬编码，可在任何机器跑；tools/ 需配置环境变量
- **依赖**: Python 3.10+, FFmpeg, edge_tts, moviepy, Pillow
- **网络要求**: edge_tts 需联网（国内需 HTTP_PROXY 代理）

## 技术决策记录

1. **配音选 edge_tts 而非 API** — 免费，质量够用，微软服务器稳定
2. **运镜用 FFmpeg 而非 MoviePy** — 平滑度差距明显，FFmpeg 是 C 实现
3. **去掉所有字幕** — 二次元画风需要沉浸感，旁白已讲清故事
4. **JSON 脚本分离** — 内容创作者不需要懂代码
5. **MD5 缓存 TTS** — 相同旁白不重复请求，省时间

## 已知限制

- edge_tts 偶发连接失败（已内置 5 次重试）
- 图片需提前生成（管线不做 AI 出图）
- 仅支持竖屏 9:16
- 无背景音乐（BGM 尚未实现）
